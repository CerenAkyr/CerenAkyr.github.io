# Ceren Akyar Portfolio

A personal portfolio site built with React and Vite.

This repo is the home for the source code, styling, and assets behind `CerenAkyr.github.io`. It is designed as a lightweight single-page portfolio with responsive sections, and a GitHub Pages deployment flow.

## Tech

- React
- Vite
- CSS
- GitHub Pages

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```

## Deploy

```bash
npm run deploy
```

The site is intentionally asset-heavy and visual, so image sizes and formats matter when updating it.
